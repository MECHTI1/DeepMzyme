## Project purpose

This repository develops DeepMzyme, a deep-learning framework for meatlloenzyme metal type prediction and  EC function prediction.

The main goals are:
1. Predict metal type from protein structural pocket graphs+ESMC models.
2. Predict from protein structural pocket enzyme class / EC-level labels .
3. Compare model variants fairly using validation and held-out test metrics.
4. Keep the code reproducible, simple to run, and suitable for publication-quality experiments.
